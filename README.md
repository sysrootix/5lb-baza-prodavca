# 5LB · База знаний продавца

Внутренний обучающий лендинг для **продавцов-консультантов** розничной сети 5LB.
Собирает в одном месте все регламенты работы на точке, чтобы новый или действующий
сотрудник мог быстро во всём разобраться.

## Что внутри

Сайт построен строго по материалам пяти внутренних документов из `docs/`:

| Раздел на сайте | Источник |
| --- | --- |
| Рабочий день · Стандарты магазина · Дисциплина · Зарплата · Особые правила | Правила работы и санкции 5ЛБ |
| Техника продаж · Общение с клиентами | Техника продаж |
| Заказы с сайта | Инструкция по работе с заказами клиентов |
| Рабочий день · SOS | Инструкция по работе в команде 5LB |
| SOS · Экстренные ситуации | Методичка: приём оплат в экстренных ситуациях |

Всего 10 разделов: рабочий день, дисциплина и санкции, зарплата и грейды,
техника продаж, общение с клиентами, касса и оплаты, заказы с сайта,
экстренные ситуации, стандарты магазина, особые правила.

## Возможности

- Адаптивная вёрстка под все экраны (мобильные, планшеты, десктоп).
- Поиск по всей базе знаний (иконка 🔍 в шапке, либо клавиша `/` или `Ctrl/⌘+K`),
  с подсветкой, навигацией по совпадениям (`Enter` / `Shift+Enter`) и
  автораскрытием аккордеонов на найденном фрагменте.
- Закрепляемая навигация со scroll-spy и индикатором прогресса чтения.
- Аккордеоны, таймлайны смены и заказа, таблицы депремирований с точными суммами,
  «лестница нарушений», карточки грейдов, блоки «можно/нельзя».
- Кликабельные Telegram-контакты техподдержки.
- Доступность: контраст по WCAG AA, видимый фокус с клавиатуры, `aria`-разметка
  аккордеонов и поиска, поддержка `prefers-reduced-motion`.

## Запуск

Статический сайт без сборки. Открой `index.html` в браузере либо подними сервер:

```bash
python3 -m http.server 8080
# открыть http://localhost:8080
```

## Структура

```
index.html              — разметка и весь контент
assets/css/styles.css   — дизайн-система, адаптив, анимации
assets/js/main.js       — навигация, поиск, scroll-spy, аккордеоны
docs/                   — исходные .docx-методички
```

## Фирменный стиль

Оранжевый `#FF6600` / `#FF8A1F` (для мелкого текста — контрастный `#C2410C`),
тёплый белый фон, тёмно-серый текст. Шрифты: **Unbounded** (заголовки) + **Inter** (текст).

## Примечание по безопасности

Код снятия с охраны филиала, фигурирующий в исходных документах, намеренно
не выводится на страницу — заменён на формулировку «по коду от руководителя».
